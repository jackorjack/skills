# 混剪-引流型 / 人设类

（1）这就是我和老公投资五十万在金华义乌创业开的一个小纸箱厂，创业第六年了，今天跟大家讲讲我们的故事。早上七点我们准时来上班了，师傅这会儿把纸板搬到机器上开始排版、印刷。我们第一次创业，从刚开始的一百五十平场地到现在的三百平，从二手印刷机到现在印刷更漂亮的一手机器，虽然变化不大，但是一切都往好的发展。虽然这不是一份体面的工作，但是工作没有高低之分，我们都是靠自己的双手挣钱。

（2）有人说不就几分钱的东西，也没必要较真。你想客户拿到衣服，摸到第一个产品就是这个袋子，这个封口没做好，印刷印糊了，都会影响客户对我们产品的信任。包装不只是附属品，更是产品价值的延伸。几分钱的袋子能装下商家的用心，也能帮客户提升口碑，这就是小物品的大价值。

（3）做全屋定制或许能短暂的欺骗一个人，但绝不可能蒙蔽一群人。一个客户满意是偶然，两个客户信任是默契，三个客户选择是口碑，无数客户认可则是对我们品质与服务的肯定。
