# 大字报格式参考示例

> 来源：抖音大字报视频截图
> 形式：画面叠加文字 + 字幕/配音 + 话题标签

---

## 结构拆解

```
┌─────────────────────────────────────┐
│  🖼️ 视频画面（产品/工厂/过程素材）    │
│                                     │
│  ┌─────────────────────────────┐    │
│  │ 🔴 牛仔裤工厂老板吓一跳：     │    │ ← 标题（大字叠加）≤35字
│  │    印个花不到2毛，           │    │   冲突感+反差数字
│  │    出口老外却要300美金！     │    │
│  └─────────────────────────────┘    │
│                                     │
├─────────────────────────────────────┤
│  老板原话：国内打价格战早卷疯了，    │ ← 文案（字幕/配音）
│  一条牛仔裤挣个块八毛。我这直接      │   说明原委+制造冲突
│  做出口，满打满算少说多赚一百倍。    │   50-80字
│  说白了赚的就是信息差的钱。          │
├─────────────────────────────────────┤
│  #牛仔裤 #数码印花 #牛仔加工          │ ← 话题标签（3-5个）
└─────────────────────────────────────┘
```

## 关键要素

| 要素 | 说明 | 要求 |
|---|---|---|
| **标题** | 画面上方大字叠加（红框/醒目色），是视频的第一眼钩子 | ≤35字，带冲突/反差/数字/好奇 |
| **文案** | 画面下方的字幕或配音文案，展开说明标题中的故事 | 50-80字，口语化，补充细节 |
| **标签** | 视频底部话题标签 | 3-5个精准标签，覆盖产品/行业/人群 |

## 标题技巧

- **反差数字**："不到2毛" vs "300美金"
- **身份冲突**："工厂老板" + "吓一跳"（反差）
- **结果导向**：直接点明利益/差距
- **好奇缺口**：让人想知道"为什么""怎么做到的"

## 文案技巧

- 第一句承接标题展开
- 用"原话""说白了"等口语词
- 制造"内行才知道"的信息差感
- 结尾点题：赚的是XX的钱
