# 真人出镜-种草型 / 应用场景

（1）设计好的机箱机柜外壳是怎么生产出来的？
首先是激光切割下料，师傅根据图纸要求编程，并选择合适的板材进行下料。
接着是折弯，将切割后的板材进行折弯加工，使板材的形状规格符合要求。
三、焊接，通过焊接将钣金件连接起来。
四、打磨，将焊接切割后的焊缝、毛刺打磨平整。
五、喷涂，根据客户需求对钣金件进行喷涂。
最后装配，检查无误后就可以打包发货了。
