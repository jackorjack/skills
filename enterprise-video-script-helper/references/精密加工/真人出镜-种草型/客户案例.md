# 真人出镜-种草型 / 客户案例

（1）昨天我们的客户拉了一车30毫米厚的q355的钢板过来，要求激光切割出来的品质第一要光滑，第二要准确。我们就选用了氧气来切割，氧气切割的特点是能达到表面光滑，但是氧气切割的速度稍微慢一点。现在30毫米厚的钢板大概1分钟能走到1.8米左右，大家来看一下现在激光出来的效果。首先看这个圆形还是比较圆润的，同时接口是没有毛质的，很光滑的。正面和反面的锥形也控制在0.1毫米之内。最重要的能帮客户解决问题了，之前他们是用火焰来切割了，火焰来切割第一效率慢，第二主要有毛刺。但是选用激光来切割，同时用氧气来切割就可以解决了客户这个问题。

（2）这是通过短视频找来的客户，他们有4000件，这个37乘以22乘241的，他原来是用激光开的，然后45号钢激光开的边比较硬，他们回去加工，刀片损耗，人工成本都高了很多。现在找到我们这4000件，用这个开，老板说每一件最少省了三块钱4000件。我给他省了一万多块钱，大家说现在制造行业那么难，赚钱不如省钱。
