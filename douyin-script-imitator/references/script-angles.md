# 脚本角度参考

仿写3条脚本时，选3个不同角度：

| # | 角度 | 说明 |
|---|------|------|
| 1 | 痛点放大 | 放大客户最痛点，再给方案 |
| 2 | 卖点直击 | 开头直接甩核心卖点 |
| 3 | 场景代入 | 用客户日常场景开头 |
| 4 | 对比碾压 | "别人 vs 我们"制造反差 |
| 5 | 数据说话 | 用具体数字/案例建立信任 |
| 6 | 故事共鸣 | 用客户案例/小故事开头 |
| 7 | 行业揭秘 | "行业内幕""避坑指南"切入 |
| 8 | 反常识 | 反直觉观点做钩子 |

选择原则：根据原视频类型匹配角度；3条覆盖不同类型；有明确优势优先放大。
