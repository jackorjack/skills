# 抖音搜索

通过 Stealth Browser 搜索抖音视频，绕过风控返回 **Markdown 表格**。自动拉取评论，**有咨询意图的视频优先展示**，**仅返回 2022 年及以后的视频**。

## 工作流程

1. 检查 `scripts/cookie.txt` 是否存在且未过期
2. 缺失或过期 → 按 `references/cookie-prompt.md` 话术索要新 Cookie
3. 搜索 → 拦截 API → 解析最多 20 条视频
4. **过滤：仅保留 2022 年及以后的视频**（create_time >= 2022-01-01）
5. 每条视频拉取前 15 条评论 → 匹配咨询关键词
6. **排序：有咨询意图优先 → 咨询命中数降序 → 点赞降序 → 取前 20 条**
7. 输出 Markdown 表格

## 运行命令

```bash
node scripts/search.js "<关键词>"
```

## 咨询关键词

覆盖询价、购买意向、联系方式、合作批发、工厂 OEM、样品起订、发货交期等 30+ 词，适配制造业 B2B 场景。详见 `scripts/search.js` 中 `CONSULTATION_KW`。

```markdown

| # | 视频标题 | 👍点赞 | 💬评论 | ⏱时长 | 👤作者 | 👥粉丝 | 📅发布日期 | 💰咨询意图 |
|---|----------|--------|--------|--------|--------|--------|------------|------------|
| 1 | [鞋垫源头工厂日产10万双](https://www.douyin.com/video/xxx) | 2.3万 | 156 | 1:30 | 振滔鞋材 | 5.2万 | 2025-12-15 | ✅ 多少钱、工厂、定制 |
| 2 | [透气鞋垫批发拿货](https://www.douyin.com/video/xxx) | 1.8万 | 89 | 0:45 | 鞋材老王 | 3.1万 | 2025-11-20 | ✅ 批发、拿货 |

> 💰 咨询意图列展示评论中匹配到的询价/购买/合作关键词
```

### 咨询意图列说明

- **✅** + 关键词列表 → 该视频评论包含询价、购买意向、合作咨询等需求
- **❌ 无** → 评论中未匹配到咨询关键词，属于普通曝光内容
- 咨询意图视频优先排序，命中关键词数越多越靠前

## 退出码

| 退出码 | 含义 |
|--------|------|
| 0 | 搜索成功 |
| 1 | 缺少关键词参数 |
| 2 | Cookie 文件缺失或为空 |
| 3 | Cookie 已过期（sid_guard 失效） |
| 4 | 触发验证码拦截 |
| 5 | 其他搜索错误 |
